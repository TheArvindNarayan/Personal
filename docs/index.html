Color Palette


Brand Color

Brand Color is Warm-Blue which is defined by SASS variable $main-colorand classes .color-blue and .background-blue.Further there is a light and a dark variation for the brand color which are defined by postfix $**-dark or $**-light and helper classes .background-blue-dark .color-blue-dark .background-blue-light .color-blue-light

Complimentary Colors

Complimentary Color are set of Colors which can be used where it's deemed needed
Green is defined by SASS variable $greenand helper classes .color-green and .background-green Red is defined by SASS variable $red and helper classes .color-red and .background-red
Yellow is defined by SASS variable $yellow and helper classes .color-yellow and .background-yellow

#2ce285

Background/Foreground Colors

Background/Foreground Colors are defined by SASS variables $black  $white and helper classes .background-white  .color-white  .background-black .color-black. 
Further there are shades of both black and white which can be accessed by sass variables $black-light  $black-dull $white-light $white-dull  and helper Classes with Postfix of .**-light .**-dull

Typography


Headings

Headings H1-H3 are by Default formated to Montserrat and Sans-Serif as Fallback Fonts Font Sizes and other parameters change automatically via Media Queries. You can useMontserrat by using .heading helper Class or SASS variable $title-font

Paragraph

The Paragraph, Anchor, Span and Rest are by Default formated to Questrial and Sans-Serif as Fallback Fonts Font Sizes and other parameters change automatically via Media Queries. You can also useMontserrat by using SASSvariable $font

Buttons

Buttons are by Default styled using class .btn


Colored

Buttons can be colored by using color palette defined above



Outlined

Buttons can be outline with respective color with .btn-outline helper class but strictly followed color palette helper class
color-* Thickness can be altered by .bdr-size-* min or max



Rounded

Buttons can be rounded with respective color and outline definations using class .rounded-bdr further more there are two other variations or curves namelyrounded-bdr-min rounded-bdr-max
